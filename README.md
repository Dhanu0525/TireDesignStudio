### Prerequisites

- Make sure you have `Node.js` and `npm` (or `yarn`) installed.

### Installation

1. Clone the repository:  git clone https://github.com/Dhanu0525/TireDesignStudio.git
2.Navigate to the project directory
   cd TireDesignStudio
3.Install dependencies:
   npm install
4.Run the project:
  npm start
